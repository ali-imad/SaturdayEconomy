# SaturdayEconomy
A simple economy plugin for our personal Minecraft server. 

## Requirements:
For use in a server, Vault, mcMMO, and Essentials


## Features:
 - Simple configurations
 - Player professions
   - Block breaking rewards
   - Mob slaying rewards
 - Salaries
    - Hourly
 - Commands
   - /tempfly -> Temporarily fly for a certain amount of time
